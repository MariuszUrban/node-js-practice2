MANROPE IS A MODERN SANS-SERIF FOR EVERYONE.

This font is a crossover of different font types: it is semi-condensed, semi-rounded, semi-geometric, semi-din, semi-grotesque. It employs minimal stoke thickness variations and a semi-closed aperture.

The fourth version of the font comes in a Variable Font format, and 7 legacy weights.

Some software automatically replaces (') with (’) in your writing, according to the context. But no internet browser is doing that. Manrope font uses the OpenType feature, which sets a correct apostrophe automatically.
